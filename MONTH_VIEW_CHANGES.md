# Month View Calendar Implementation

## Изменения

Добавлена календарная сетка для month view с следующими возможностями:

### Функциональность

1. **Календарная сетка 7x6** - отображает месяц в виде календаря с днями недели
2. **Кликабельные дни** - клик по дню переключает на day view этого дня
3. **Индикация задач** - дни с задачами помечены синей точкой и жирной рамкой
4. **Выходные дни** - суббота и воскресенье выделены темным фоном
5. **Праздничные дни** - российские праздники 2025 года выделены красноватым фоном
6. **Текущий день** - выделен синей рамкой
7. **Навигация по месяцам** - кнопки "Prev Month" / "Next Month"

### Визуальные индикаторы

- **Обычный день**: светло-серый фон (#252526)
- **Выходной день**: темно-серый фон (#2a2a2d)
- **Праздничный день**: красноватый фон (#3a2a2d)
- **День с задачами**: синяя рамка (#4fc1ff) + синяя точка в правом нижнем углу
- **Текущий день**: двойная синяя рамка + голубой фон (#1e3a4f)
- **Дни другого месяца**: полупрозрачные (opacity: 0.3)

### Праздничные дни 2025

Жестко закодированы в функции `isHoliday()`:
- Новогодние каникулы: 1-8 января
- День защитника Отечества: 23-24 февраля
- Международный женский день: 8-10 марта
- Праздник Весны и Труда: 1-4 мая
- День Победы: 9-11 мая
- День России: 12-14 июня
- День народного единства: 3-4 ноября

### Технические детали

1. **Файл**: `src/views/agendaPanel.ts`
2. **Новые функции**:
   - `renderMonthCalendar(days)` - рендеринг календарной сетки
   - `isHoliday(date)` - проверка праздничных дней
   - `navigateToDay(date)` - переход на day view
3. **Обработка кликов**: при клике на день отправляется сообщение с флагом `switchToDay: true`
4. **Переключение view**: панель закрывается и открывается новая с day view

## Использование

1. Откройте Agenda Month: `Ctrl+Shift+P` → "Markdown Org: Show Agenda (Month)"
2. Кликните на любой день для просмотра задач этого дня
3. Используйте кнопки навигации для перехода между месяцами
